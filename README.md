# photoswiper

[![lifecycle](https://img.shields.io/badge/lifecycle-experimental-orange.svg)](https://www.tidyverse.org/lifecycle/#experimental)

The goal of photoswiper is to create a simple api for adding an image
gallery to a `shiny` application. [PhotoSwipe](http://photoswipe.com/)
is a minimalist javascript gallery with no dependancies, created by
[Dimitri Semenov](https://github.com/dimsemenov).

## Installation

You can install the released version of photoswiper from
[github](https://github.com/mrjoh3/photoswiper) with:

``` r
devtools::install_github("mrjoh3/photoswiper")
```
